main:   A-B-C-D----------- 
             \           /
my-branch:    A-B-C-F-G-H 


old: A-B-C 
my new commits: F-G-H
others commit on main: D

rebase: old + D + my-new-commits
git pull origin main --rebase

After pull --rebase :

main:   A-B-C-D----------- 
             \             /
my-branch:    A-B-C-D-F-G-H 


After merge "my-branch" into main:

main:         A-B-C-D-F-G-H
my-branch:    A-B-C-D-F-G-H

we can delete our temp branch:
git branch -D my-branch
